# Rough GDD

Players form a small co-op crew and enter a dream zone. Each run asks the
players to collect memory shards, avoid escalating manifestations, and reach an
escape portal before the collapse timer expires.

Candidate systems:

- Dream zone run loop.
- Memory shard collection.
- Manifestation enemy pressure.
- Escape portal objective.
- Upgrade loop between runs.
- Contract board for optional objectives.

Unspecified:

- Exact target platforms.
- Unity version.
- Networking model.
- Save model.
- Telemetry payloads.
- Liveops expectations.
- Monetization.
