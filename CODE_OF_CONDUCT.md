# Code of Conduct

VerityFoundry follows a simple contribution standard: be respectful, direct,
and constructive.

## Expected Behavior

- Focus on technical merits and user outcomes.
- Preserve uncertainty in prompt workflows.
- Treat contributors and users with respect.
- Avoid harassment, personal attacks, and discriminatory language.

## Reporting

Report conduct concerns through the repository maintainers using the contact
paths in `SUPPORT.md`. Maintainers may remove comments, issues, pull requests,
or contributors that violate this standard.
