# Readiness Targets

VerityFoundry readiness targets are authoring targets, not final certification
claims.

Supported targets:

- `concept-complete`
- `design-complete`
- `prototype-ready`
- `vertical-slice-ready`
- `implementation-ready`
- `production-ready`
- `operations-ready`
- `liveops-ready`
- `maintenance-ready`
- `decommission-ready`
- `archival-ready`

Generated outputs should include readiness gaps. VeritySpec should validate
the resulting workspace.
