# Interview Modes

VerityFoundry supports five interview modes.

| Mode | Behavior |
|---|---|
| `interview-none` | Ask no questions. Generate a conservative low-stakes candidate and mark assumptions. |
| `interview-low-stakes` | Ask a small number of high-value questions for concept and prototype work. |
| `interview-medium-stakes` | Ask architecture, implementation, telemetry, and handoff questions. |
| `interview-high-stakes` | Ask before legal, financial, privacy, compliance, platform, monetization, liveops, or data decisions. |
| `interview-all` | Ask everything required to minimize assumptions. |

When uncertain, leave decisions unresolved rather than fabricating certainty.
