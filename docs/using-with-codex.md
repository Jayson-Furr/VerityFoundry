# Using With Codex

1. Start Codex in the project that needs a candidate VeritySpec workspace.
2. Provide the rendered VerityFoundry prompt.
3. Provide source materials such as briefs, GDDs, README files, or image notes.
4. Ask Codex to preserve assumptions, provenance, unresolved questions, and
   human approval requirements.
5. Run VeritySpec validation on generated workspaces.

Repository agents working inside VerityFoundry should follow `AGENTS.md`.
